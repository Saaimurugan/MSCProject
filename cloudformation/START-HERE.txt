================================================================================
TROUBLESHOOTING
================================================================================

If deployment fails:
  1. Check AWS credentials: aws sts get-caller-identity
  2. View CloudFormation events in AWS Console
  3. Check the README.md for detailed troubleshooting

If frontend doesn't load:
  1. Check S3 bucket policy allows public read
  2. Verify .env file has correct API URL
  3. Check browser console for errors

If API returns errors:
  1. Check Lambda logs in CloudWatch
  2. Verify CORS headers in responses
  3. Test endpoints with curl or Postman
